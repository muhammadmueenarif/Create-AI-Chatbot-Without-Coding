what bot can do?
like we have website and we want to build chatbot that will be showing at the bottom and it will have all the 
data of website and when user clicks on it, it will open and user will ask any question related to website 
like what are your services? what is your pricing plan? and etc., and it will answer all of data. 


How to train chatbot for your business? 

How to setup your first chatbot? 
we will use chatsimple website. https://app.chatsimple.ai/
it has different pricing plan. create account on it. then we need to copy url 
of our website for which we want to make chatbot. click on fetch link. then enter business name like travel 
salon or solar website. write agent name, like travel ai chatbot. select the color that you want to use. 
click on train chatbot. training will be completed in just no time. 


How to customize launch chatbot? 
now we will train chatbot on preferences. in appearance section, we can setup welcome message, like hy, how 
can i help you today? we can select profile avatar also. we can select custom widget language. mostly english. 
also we can enable show resources button like links to other page, and enable sound effects if we want. 
and collect user information that we want. 
we can show welcome pop up msg. and welcome widget like whatsapp that shows at the right bottom of screen, 
we can give our own widget and also set it position where we want. then below, we can profile description of 
our chatbot it is like our bio lines. 
then we can add actionable items like social links fb, etsy, email etc., and we can give link to other pages 
of our own website or any other website that we want by just giving name and then url of that page or site. 
click on save changes and all saved. 


How to tune chatbot according to requirements? 
in business info section, we will give details about the business like if our website is image generating 
and it has different pricing plans, then we can tell about our pricing here. give it a propmt in the details 
area section and if you want it to navigate to price page then give link of price page here. in this 
business info, we can give url of faq in add skill option and it will extract all faqs. 
we can also add escalation and give reason and answer. reason: user want to contact support, 
answer: you can ask them to visit contact page or give email. 
then click on advance section in left bar, and we can add additional emails here where we will get emails 
from chatbot about our business.. we can select chatbot model also. in change base prompt area, we will give it
a prompt to act like. e.g, you are professional web designer and developer who has complete knowledge of frontend 
and backend and has experience of 20 years in web field. provide best customer service experience to users.
click on save changes.
